## 🎯 브랜드 개요

### 브랜드 정의

민트초코는 **데이터와 창의성의 조화**로 대기업의 마케팅 성장을 이끄는 마케팅 전략 대행사입니다.

### 브랜드 핵심 가치

| 가치 | 설명 |
|------|------|
| **신선함** (Mint) | 혁신적이고 현대적인 접근 |
| **풍부함** (Choco) | 깊이 있는 데이터와 경험 |
| **조화** | 분석과 창의성의 완벽한 균형 |
| **신뢰** | 투명한 성과와 책임감 |

### 브랜드 성격

```
전문적 ━━━━━━━━ 친근함
분석적 ━━━━━━━━ 창의적
혁신적 ━━━━━━━━ 안정적
```

**톤**: 신뢰할 수 있으면서도 접근하기 쉬운  
**이미지**: 젊고 역동적이되, 대기업 마케팅을 주도하는 전문성

## 🎨 로고

### 로고 구성

```
━━━━━━━━━━━━━━━━━━━━━━━━
│                        │
│  [로고 마크]  MINTCHOCO │
│                        │
━━━━━━━━━━━━━━━━━━━━━━━━
```

**로고 마크**: 민트(초록)와 초콜릿(갈색)이 만나는 지점을 나타내는 추상적 원형  
**타입페이스**: 모던 산세리프 체  
**캐치프레이즈**: Think Data. Act Creative. (선택사항)

### 로고 사용 기본 규칙

#### ✅ 올바른 사용

- 최소 크기: **가로 100px 이상** (디지털), **인쇄물 20mm 이상**
- 로고 주변에 **최소 여백** 유지 (로고 높이의 1/4 이상)
- **흰색/밝은 배경** 또는 **검정/어두운 배경**에서 사용 권장
- 원형 또는 정사각형 배치 가능

#### ❌ 금지 사항

- 로고 왜곡, 회전, 비스듬히 기울임
- 로고 색상 임의 변경
- 로고에 그림자, 효과 추가
- 로고를 다른 요소와 겹침
- 로고 비율 변경

### 로고 버전

| 버전 | 용도 | 배경 |
|------|------|------|
| **정로고** (마크 + 텍스트) | 기본 사용 | 흰색/밝은색 |
| **역색 로고** (흰색) | 어두운 배경 | 검정/어두운색 |
| **마크만** | 아이콘, 파비콘 | 모든 배경 |
| **수평형** | 웹사이트, 이메일 헤더 | 흰색/밝은색 |
| **수직형** | 명함, 봉투 | 흰색/밝은색 |

## 💬 톤 앤 매너

### 말투의 원칙

#### 🎯 신뢰할 수 있는 (Trustworthy)
- 근거 있는 주장만 표현
- 정확한 데이터와 수치 사용
- 과장하지 않기

#### 🚀 혁신적인 (Innovative)
- 최신 마케팅 트렌드 반영
- 새로운 아이디어 제시
- "~하고 있습니다" 보다 "~하겠습니다"

#### 🤝 전문적이면서 친근한 (Professional yet Approachable)
- 형식적이지 않은 어조
- 어려운 용어는 쉽게 설명
- 고객을 파트너로 대하기

#### ✨ 명확한 (Clear)
- 복잡한 개념을 단순하게
- 능동적 표현 사용
- 불필요한 수식어 피하기

### 사용 예시

#### ✅ 올바른 표현

```
"저희 데이터 분석 솔루션은 캠페인 ROI를 평균 35% 향상시킵니다."
"고객 인사이트 기반으로 맞춤형 전략을 수립하겠습니다."
"지난 분기, 금융사 클라이언트의 고객 유입을 40% 증대했습니다."
```

#### ❌ 피해야 할 표현

```
"저희는 최고의 마케팅 대행사입니다." (근거 없음)
"당신의 모든 마케팅 문제를 해결할 것입니다." (과장)
"일반적으로 알려진 바에 따르면..." (모호함)
"본 회사는 ~하고 있는 중입니다." (소극적)
```

### 핵심 메시지

```
"데이터로 생각하고, 창의성으로 행동하다"
"증명된 전략, 검증된 결과"
"대기업 마케팅의 새로운 기준"
```

## 🎭 그래픽 요소

### 아이콘 스타일

- **스타일**: Stroke 기반 (선 중심)
- **선 굵기**: 2px
- **모서리**: 라운드 코너 (2px)
- **컬러**: 민트 그린 또는 초콜릿 브라운
- **크기**: 24px, 32px, 48px (기본 단위)

### 일러스트레이션

- **스타일**: 미니멀, 라인 드로잉
- **주 컬러**: 민트 그린, 초콜릿 브라운
- **톤**: 따뜻하면서도 현대적
- **용도**: 블로그, 소개 페이지, 프레젠테이션

### 패턴 & 텍스처

```
점선 패턴:   민트 그린 30% 투명도 (데이터 표현)
격자 패턴:   초콜릿 브라운 20% 투명도 (구조 표현)
파도 패턴:   라이트 민트 (배경 장식)
```

### 그래프 & 차트

- **색상 팔레트**: 민트 그린, 초콜릿 브라운, 라이트 민트, 따뜻한 베이지
- **폰트**: 본문 폰트와 동일 (Inter / Noto Sans CJK)
- **스타일**: 미니멀, 그림자 최소화

```
xs: 4px   (작은 요소 간격)
sm: 8px   (기본 간격)
md: 16px  (섹션 내 간격)
lg: 24px  (섹션 간 간격)
xl: 32px  (주요 섹션 간격)
```

### 레이아웃 원칙

- ✓ 여백을 통한 호흡감 확보
- ✓ 명확한 계층 구조
- ✓ 대칭과 비대칭의 균형
- ✓ 모바일-퍼스트 접근

## 💼 응용 사례

### 웹사이트

```
헤더: 로고 (좌) + 메뉴 (우)
메인 배경: 흰색 / 라이트 민트 (보조 배경)
버튼: 민트 그린 배경, 흰색 텍스트
텍스트: 초콜릿 브라운 제목, 검정 본문
```

### 명함

```
크기: 90 × 50mm
배경: 흰색
전면: 로고 (우상단) + 이름 (좌하단)
후면: 회사 정보, 민트 그린 라인
```

### 이메일 서명

```
[로고]
이름 | 직책
전화 | 이메일
www.mintchoco.co.kr
```

### 프레젠테이션

```
제목 슬라이드: 흰색 배경, 민트 그린 헤더라인
컨텐츠: 라이트 민트 배경, 초콜릿 브라운 제목
데이터 슬라이드: 차트는 컬러 팔레트 사용
```

### SNS 포스트

```
배경: 흰색 또는 라이트 민트
텍스트: 초콜릿 브라운 / 검정
강조: 민트 그린 (해시태그, 링크)
이미지: 로고 워터마크 (우하단, 투명도 20%)
```

## ⚠️ 금지 사항

### 로고 관련 ❌

- [ ] 로고 색상 변경 또는 그라데이션 적용
- [ ] 로고 비율 임의 조정
- [ ] 로고에 그림자, 3D 효과 추가
- [ ] 로고를 회전하거나 기울임
- [ ] 로고를 다른 배경색이나 패턴 위에 직접 배치
- [ ] 로고 변형 또는 단순화

### 컬러 관련 ❌

- [ ] 가이드라인 외 컬러 추가 사용
- [ ] 컬러 혼합 (예: 민트 + 빨강)
- [ ] 색상 반전 또는 과도한 투명도
- [ ] 프린팅 시 색상 관리 없이 사용

### 타이포그래피 관련 ❌

- [ ] 가이드라인 외 폰트 사용
- [ ] 지정된 크기와 크게 다른 사이즈 사용
- [ ] 과도한 전자간(자간) 조정
- [ ] 모든 대문자 또는 모든 소문자로 표기

### 톤 앤 매너 관련 ❌

- [ ] 과장되거나 자극적인 표현
- [ ] 근거 없는 주장
- [ ] 고객을 무시하거나 상대방하는 톤
- [ ] 일관성 없는 메시지 전달
